# SubQ Documentation

Documentation site for the SubQ Speech-to-Text API, built with Fumadocs and Next.js.

## Typography

Style guide: **large numbers** (Roboto Mono Light) — use Tailwind classes such as `font-mono font-light tabular-nums` with your chosen size (e.g. `text-4xl`).

## Development

1. **Install dependencies**:

   ```bash
   npm install
   ```

2. **Generate Source Files**:

   ```bash
   npx fumadocs-mdx
   ```

3. **Start development server**:

   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## Deployment

This boilerplate is a standard Next.js application and can be deployed to any platform that supports Next.js, such as Vercel, Netlify, Railway, or self-hosted environments.
