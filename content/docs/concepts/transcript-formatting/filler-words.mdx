---
title: Filler words
description: Include filler words in transcription with filler_words=true
---

The filler words feature controls whether filler words like "um", "uh", "like", and "you know" appear in the transcript. This is useful for verbatim transcription or conversation analysis where hesitations carry meaning.

## Usage

Add `filler_words=true` as a query parameter to include filler words in the output:

```bash
curl -X POST "https://api.aldea.ai/v1/listen?filler_words=true" \
  -H "Authorization: Bearer YOUR_ALDEA_API_KEY" \
  --data-binary @audio.wav
```

## Example

| `filler_words` | Transcript |
|---|---|
| `false` (default) | "I think we should move forward with the project." |
| `true` | "I think, um, we should, you know, move forward with the project." |

<Callout type="info">
Filler word detection is available for English only.
</Callout>
